PLUGIN_CUSTOMSKIP
	EN	Custom Skip

PLUGIN_CUSTOMSKIP_DESC
	EN	Custom Skip lets you define client-specific filter sets with rules for when tracks should be skipped.

PLUGIN_CUSTOMSKIP_SETTINGS
	EN	Settings

SETTINGS_PLUGIN_CUSTOMSKIP_PARENTFOLDERPATH
	EN	<u>Parent</u> folder for filter set directory

SETTINGS_PLUGIN_CUSTOMSKIP_PARENTFOLDERPATH_DESC
	EN	Set the <b>parent</b> folder for the automatically created <b>CustomSkip</b> folder where the plugin stores the filter set files.

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_ENABLED
	EN	Enable look-ahead filtering

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_ENABLED_DESC
	EN	If enabled <i>Custom Skip</i> will check not only the current track, but also the following tracks to see if they have to be skipped/removed from the playlist.

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_RANGE
	EN	Number of subsequent tracks to check

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_RANGE_DESC
	EN	Set the number of subsequent tracks to check after the current track.<br><br><b>Please note:</b> <i>Custom Skip</i> will always leave at least one last track in the playlist after the currently playing track to avoid problems with plugins that subscribe to <i>newsong events</i> (= song change triggers some action).

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_DELAY
	EN	Look-ahead delay

SETTINGS_PLUGIN_CUSTOMSKIP_LOOKAHEAD_DELAY_DESC
	EN	Set the time CustomSkip waits after a song change event before starting the look-ahead filtering process.

SETTINGS_PLUGIN_CUSTOMSKIP_JIVEMENU_CHANGEPRIMARYFILTERSET_ENABLED
	EN	Enable filter set change menu<br>on jivelite players

SETTINGS_PLUGIN_CUSTOMSKIP_JIVEMENU_CHANGEPRIMARYFILTERSET_ENABLED_DESC
	EN	This will enable a menu on jivelite players (piCorePlayer, SB Touch, SB Radio, SqueezePlay...) that lets you easily change the <b>active primary</b> filter set for your client or disable all filtering.
